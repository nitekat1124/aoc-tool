# aoc-tool

a simple tool to create Advent of Code templates

```
usage: app.py [-h] [-y year_number] [-t target_dir] [-p dir_prefix]

Advent of Code Tool

options:
  -h, --help            show this help message and exit
  -y year_number, --year year_number
                        the year of the AoC event
  -t target_dir, --target target_dir
                        target directory for the AoC files
  -p dir_prefix, --prefix dir_prefix
                        prefix for the AoC files directory
```
